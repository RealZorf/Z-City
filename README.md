## About

VOTTUR'S ZCITY is a Garry’s Mod server built around immersive systems, custom gameplay mechanics, and a continuously evolving player experience.

Core focus:
- Realistic, responsive in-game interactions
- Community-shaped gameplay and development
- Custom systems, mechanics, and server-wide features

## What is Z-City

Z-City is a Garry’s Mod addon that modifies character damage and movement controls.  
It also includes its own weapon base and gamemode framework.

## Modules

### 8bit Module

https://github.com/uzelezz123/8bit_zcity

The compiled version is included in `lua/bin`.

If the module does not function correctly, move the `.dll` files directly into: `garrysmod/lua/bin` (instead of leaving them inside the addon folder).

This module is required for voice chat manipulation features, such as Combine voice effects.

### Optional: Discord Rich Presence

- https://github.com/YuRaNnNzZZ/gmcl_steamrichpresencer/releases/tag/2023.07.20  
- https://github.com/fluffy-servers/gmod-discord-rpc/releases/tag/1.2.1  

These modules enable Discord RPC integration for clients.
